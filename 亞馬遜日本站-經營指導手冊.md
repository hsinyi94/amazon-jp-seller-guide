### 【03】成本預估 💰

> 📌 **本章重點：** 日本站主要成本 = 月費 + 銷售傭金 + FBA 物流倉儲費 + 廣告費 + 促銷費。用 FBA 計算機估算你的實際成本。

**亞馬遜銷售成本一覽：**

| # | 費用項目 | 說明 |
|---|---------|------|
| 1 | 專業帳戶月費 | 每月 ¥4,900（未申報統編將收 5% 營業稅 = ¥5,145） |
| 2 | 成交手續費（銷售傭金） | 依[商品類別](https://sellercentral.amazon.co.jp/help/hub/reference/external/GT23F3ST7FEZKT9A?locale=zh-TW&initialSessionID=356-3357753-9849531&ld=SOTWSOAIGLOGISTICSNULLMKTJPFeeChange251208)收取成交金額的 8~15% |
| 3 | FBA 物流 + 倉儲費 | 日本亞馬遜倉 → 消費者的物流費 + 月度倉儲費 |
| 4 | 站內廣告費 | 透過設定預算上限控制。建議新品至少投資 10~20% 銷售額於廣告 |
| 5 | 站內促銷費 | 透過設定預算上限控制 |

**✅ 行動項目：**
- [ ] 使用 [FBA 計算機](https://sellercentral.amazon.co.jp/hz/fba/profitabilitycalculator/index?lang=zh_TW) 估算 FBA 物流 + 倉儲費成本
- [ ] ▶️ [FBA 計算機使用方式說明（YouTube）](https://www.youtube.com/watch?v=rOqNx-bzMG8)

### 【04】產品合規 🔌

> 📌 **本章重點：** 日本站有特殊合規要求。電器需 PSE/Giteki，嬰幼兒/食品/美妝需 JFSL。務必在上架前確認。

**⚠️ 你的商品需要哪些合規？快速判斷：**

| 商品類型 | 需要的合規 | 提交方式 |
|---------|-----------|---------|
| 電器商品 | PSE、METI | 上架後 email 至 jp-electronics-safety@amazon.co.jp |
| 藍牙 / WiFi 商品 | Giteki (Telec) | 上架後 email 至 jp-giteki-safety@amazon.co.jp |
| 嬰幼兒商品 | 日本食品衛生法 (JFSL) | 依商品類別提交 |
| 食品 / 接觸食品的容器 / 美妝品 | JFSL | 依商品類別提交 |
| 其他不確定 | 請諮詢亞馬遜官方經理 | — |

**合規準備步驟：**
1. 確認產品是否需要合規檢驗
   - 電器商品：PSE [審核規範](https://sellercentral.amazon.co.jp/help/hub/reference/external/GHJPQ6KDLA9N5F4L?locale=zh-TW)（附件：Amazon JP_PSE 申請範本.docx）
   - 藍牙/WiFi：Giteki [審核規範](https://sellercentral.amazon.co.jp/help/hub/reference/external/GV3N5ES8KNR3W3VG?locale=zh-TW)（附件：Amazon JP_Giteki 申請範本.docx）
   - 食品飲料：[有效期產品進倉規範](https://sellercentral.amazon.co.jp/help/hub/reference/external/201003420?locale=zh-TW) — FBA 要求庫存接收時最低剩餘架儲期超過 60 天；距到期日 45 天內的產品將被標記為報廢
2. 針對需要合規檢驗的產品準備相關報告與文件
3. 提交文件到亞馬遜後台通過上架審核
4. 提交文件給頭程物流服務商在進口報關審核用

### 【10】促銷

> 📌 **本章重點：** 新賣家建議先開優惠券（門檻低），等排名穩定再用其他促銷工具。日本站獨有「積分」制度別忘了設定。

**給新賣家的建議：**
1. 關注亞馬遜官方經理發出的官方促銷活動 mail（亞馬遜會做站外曝光，人流比一般時段多），並調整廣告及促銷預算集中在活動期間
2. 利用回饋先開啟[優惠券](https://m.media-amazon.com/images/G/28/TWGS/SU/GN_GS_Promotions_2.2_Coupons_TW.pdf?initialSessionID=355-3002025-1543835&ld=IRTWSOA30D)（優勢：提報門檻低，要求的折扣幅度低），等排名上升有穩定銷量再提報其他促銷工具

**🔗 學習資源：**
